How to Run This React + Vite Project 
Download the Project
Download the project ZIP from GitHub.
Extract the ZIP file to a folder on your computer.
# Open the Project in VS Code
Open Visual Studio Code.
Go to File → Open Folder.
Select the extracted project folder.
# Install Node.js and npm
Download Node.js from https://nodejs.org
Install the LTS version.
This also installs npm.
# To verify installation, open a terminal and type:
# node -v
# npm -v
Open VS Code Terminal
In VS Code, go to Terminal → New Terminal.
Install Project Dependencies
# Run the following command in the terminal:
# npm install
This installs all required libraries from package.json.

# Install React Router DOM (if not already installed)
Run:
# npm install react-router-dom
# Install GSAP (if not already installed)
Run:
# npm install gsap

GSAP is used to create animations in the project.

# Run the project with:
npm run dev

Vite will start and show a URL like:
http://localhost:5173/

Open the Project in Your Browser
